# Glossary

Key terms and definitions used throughout this project.


| Term | Definition |
|------|------------|
| **CheckResult** | Result of running a loop check. |
| **LoopStore** | Manages loop entries as JSON files with fcntl locking for thread safety. |
| **Notifier** | Routes notifications to file, webhook, or stderr. |
| **SchedulerThread** | Background daemon thread that polls the store and fires checks. |
| **gen-loop-mcp** | Standalone self-scheduling follow-up MCP server — no external dependencies |
